# @etcdigital/colors-converter

## HEX ➝ RGB  
Entrada: `hexToRgb("#ff5733")`  
Saída: `{ r: 255, g: 87, b: 51 }`  

## RGB ➝ HEX  
Entrada: `rgbToHex(255, 87, 51)`  
Saída: `"#ff5733"`  

## RGB ➝ CMYK  
Entrada: `rgbToCmyk(255, 87, 51)`  
Saída: `{ c: 0, m: 0.6588, y: 0.8, k: 0 }`  

## CMYK ➝ RGB  
Entrada: `cmykToRgb(0, 0.6588, 0.8, 0)`  
Saída: `{ r: 255, g: 87, b: 51 }`
